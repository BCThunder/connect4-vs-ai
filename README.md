# connect4-vs-ai
Our project for CPSC 481, which uses a modified version of the Connect 4 project by Shaun Halverson and our implementation for an AI oponent.

## How to Run the Game
1. Clone the repository
2. Run main.py

## File Breakdown
board.py: Manages displaying and modifying the board state, and checks if there's a winner
cpu_logic.py: Handles picking an optimal move for the cpu opponent
main.py: Handles the main game flow
player.py: Handles player input
